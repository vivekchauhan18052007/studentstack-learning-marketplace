# StudentStack – E-Commerce Learning Marketplace

A modern E-Commerce Learning Marketplace built using HTML, CSS and JavaScript with authentication, filters, cart system, dark mode and localStorage support.

---

## 🚀 Live Demo

🌐 studentstack-learning-marketplace.netlify.app

---

## 📌 About Project

StudentStack is a modern frontend e-commerce platform where students can explore and purchase notes, projects, templates and development resources.

Built using HTML, CSS and JavaScript with authentication system, cart functionality and responsive UI.

---

## ✨ Features

### 🔐 Authentication
- User Registration & Login
- Remember Me Functionality
- Auth Guard Protection
- Logout System

### 🏠 Home Page
- Product Listing
- Search Functionality
- Category Filters
- Price Range Filter
- Free / Paid Filter
- Rating Filter
- Sorting System
- Product Detail Modal
- Dark / Light Mode

### 🛒 Cart System
- Add To Cart
- Remove From Cart
- Quantity Increase / Decrease
- Order Summary
- ₹100 Auto Discount
- Coupon Code Input
- Checkout System
- Deal Of The Day

### 📦 Cart Page
- Separate Cart Page
- Secure Checkout UI
- Instant Download Badge
- 24/7 Support Badge

### 💾 Storage
- localStorage Based Persistence
- Cart Data Saved Automatically

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage API
- Font Awesome
